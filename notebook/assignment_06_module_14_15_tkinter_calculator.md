## Decisions and Rationale

- This is scaffolded as a project assignment because the PDF asks for a working project folder rather than separate small scripts.

- Validation matters because calculator apps commonly fail on empty input, repeated operators, or division by zero.
- The folder should remain self-contained so it can be zipped directly.
