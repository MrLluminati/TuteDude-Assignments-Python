## Project

## Features

- Tkinter GUI using an entry display and button grid.
- Digits, decimal input, sign toggle, backspace, clear, and equals.
- Addition, subtraction, multiplication, and division.
- Division-by-zero error handling.
- Separate calculator logic so behavior can be tested without opening the GUI.
