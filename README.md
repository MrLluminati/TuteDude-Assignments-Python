## Running the Programs

Open a terminal in this repository and run:

```powershell
python .\assignments\assignment_01_basic_python_concepts\task_1_basic_math_operations.py
python .\assignments\assignment_01_basic_python_concepts\task_2_personalized_greeting.py
```

If `python` is not available on your system path, try `py` instead:

```powershell
py .\assignments\assignment_01_basic_python_concepts\task_1_basic_math_operations.py
py .\assignments\assignment_01_basic_python_concepts\task_2_personalized_greeting.py
```

## Notes

- Assignment scripts are grouped by assignment number.
- Documentation and assignment references are kept outside the submission folder.
- Generated ZIP files should be kept in `submissions/` and are ignored by Git.
