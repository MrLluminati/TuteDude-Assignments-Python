# Assignment 11 - Module 21 OpenCV Practicals Resubmission

This is the resubmission folder for Assignment 11.

The earlier submission had basic image operations only. This resubmission adds the missing OpenCV practicals mentioned in the feedback, including video processing, camera capture, rotation, thresholding, and affine transformation.

## Note

The assignment page available to me did not include a clear practical-wise instruction list. This resubmission has therefore been prepared according to the feedback received on the previous submission.

## Practicals included

1. Read image
2. Resize and crop image
3. Rotate image
4. Threshold image
5. Affine transformation
6. Video processing with cv2.VideoCapture and cv2.VideoWriter
7. Camera capture with cv2.VideoCapture
8. Final testing
