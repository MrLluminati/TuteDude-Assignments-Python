# Assignment 1: Basic Python Concepts

This folder contains both required scripts for Assignment 1.

## Files

| File | Purpose |
| --- | --- |
| `task_1_basic_math_operations.py` | Performs addition, subtraction, multiplication, and division on two user-provided numbers. |
| `task_2_personalized_greeting.py` | Builds a full name from user input and prints a personalized greeting. |

## Run

From the repository root:

```powershell
python .\assignments\assignment_01_basic_python_concepts\task_1_basic_math_operations.py
python .\assignments\assignment_01_basic_python_concepts\task_2_personalized_greeting.py
```
